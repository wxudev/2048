# 2048 小游戏（Unity 版）

> 个人练习项目，复现了经典的 2048 核心玩法，并在开源代码基础上补充了计分、计时和重置功能。

## 📋 项目介绍

这是一个基于 Unity 引擎开发的 2048 益智小游戏。主要实现了 4x4 棋盘的滑动合并逻辑，并补充了计分、最高分保存、计时、游戏结束弹窗和重新开始功能。

## 🎮 玩法与操作

- **控制方式**：键盘方向键（↑ ↓ ← →）控制方块滑动。
- **计分规则**：两个相同数字合并，**得分加上合并后的数值**。
- **目标**：尽可能合并出更大的数字（如 1024、2048）。

## 🛠️ 技术栈

- **游戏引擎**：Unity 2021.3.45
- **编程语言**：C#
- **UI 系统**：UGUI
- **动画插件**：DOTween（滑动合并的动画效果）

## ✨ 主要功能（自己做的工作）

- **开局逻辑优化**：开局时自动生成 2 个初始方块，更符合原版 2048 的体验。
- **计分系统**：每次合并都会累加分数，并在屏幕上实时更新。
- **最高分保存**：退出游戏后，最高分会自动保存在本地，下次打开依然存在。
- **游戏计时**：从开局开始计时，并在右上角显示当前的用时（格式：00:00）。
- **游戏结束面板**：无法继续滑动时，弹出结束面板，显示本局得分和用时。
- **重新开始**：点击结束面板上的“重新开始”按钮，清空所有数据，生成新的方块开始新对局。
